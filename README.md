# Demo Task p.2

Задание заключается в том, чтобы на основе предоставленной HTML и CSS вёрстки создать веб-приложение на Laravel.

Необходимо:
•	Внедрить все HTML-страницы в проект с использованием Blade-шаблонов.
•	Вынести Navbar и Footer в отдельные компоненты для повторного использования.
•	Настроить маршрутизацию страниц через Laravel.
•	Сохранить внешний вид и структуру всех элементов (поля, кнопки, карточки) в точности как в предоставленной вёрстке.
